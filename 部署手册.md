# 基于DeepSeek的WPS作业智能批阅与分层推送决策系统

## 部署手册

### 一、环境要求

- **操作系统**：Windows / Linux / macOS
- **Python**：3.8 或更高版本
- **内存**：建议 4GB 以上
- **磁盘空间**：建议 1GB 以上可用空间

### 二、安装步骤

#### 2.1 克隆或下载代码

```bash
git clone https://github.com/yourusername/it_homework_system.git
cd it_homework_system
```

#### 2.2 创建虚拟环境（推荐）

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/macOS
python3 -m venv venv
source venv/bin/activate
```

#### 2.3 安装依赖

```bash
pip install -r requirements.txt
```

主要依赖包：
- Flask - Web框架
- python-docx - Word文档解析
- openpyxl - Excel文档解析
- python-pptx - PPT文档解析
- requests - HTTP请求

#### 2.4 初始化数据库

系统首次启动时会自动初始化数据库，无需手动操作。

#### 2.5 启动系统

```bash
python app.py
```

系统默认运行在 http://127.0.0.1:8080

### 三、生产环境部署

#### 3.1 使用 Gunicorn（Linux）

```bash
pip install gunicorn
gunicorn -w 4 -b 0.0.0.0:8080 "app:create_app()"
```

#### 3.2 使用 Waitress（Windows）

```bash
pip install waitress
waitress-serve --port=8080 --host=0.0.0.0 app:create_app
```

#### 3.3 Nginx 反向代理配置

```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://127.0.0.1:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    location /static {
        alias /path/to/it_homework_system/app/static;
        expires 30d;
    }
}
```

### 四、配置说明

#### 4.1 AI服务配置

管理员登录后，进入"系统设置"→"AI接口配置"进行配置：

**支持的AI服务商：**
- DeepSeek
- 通义千问（阿里云）
- 智谱AI（GLM）
- Moonshot（Kimi）
- 自定义OpenAI兼容接口

**配置项：**
- API Key：从对应服务商获取
- API地址：服务商提供的接口地址
- 模型名称：如 deepseek-chat、qwen-turbo 等

#### 4.2 环境变量配置（可选）

可以设置以下环境变量覆盖默认配置：

```bash
# 密钥（生产环境必须修改）
export SECRET_KEY="your-secret-key-here"

# AI配置（可选，也可在系统设置中配置）
export DEEPSEEK_API_KEY="your-api-key"
export DEEPSEEK_API_URL="https://api.deepseek.com/chat/completions"
export DEEPSEEK_MODEL="deepseek-chat"
```

Windows 系统使用：
```cmd
set SECRET_KEY=your-secret-key-here
set DEEPSEEK_API_KEY=your-api-key
```

### 五、数据备份与恢复

#### 5.1 备份

系统数据存储在 `data/app.db`（SQLite数据库），直接复制该文件即可备份：

```bash
# 备份
cp data/app.db backup/app_$(date +%Y%m%d).db

# 同时备份上传的文件
cp -r data/uploads backup/uploads_$(date +%Y%m%d)
```

#### 5.2 恢复

```bash
# 停止系统
cp backup/app_20240101.db data/app.db
# 重新启动系统
```

### 六、常见问题

#### Q: 端口被占用？

修改 `app.py` 中的端口号：
```python
app.run(host='0.0.0.0', port=8080, debug=True)  # 修改 port 参数
```

#### Q: 如何修改默认管理员密码？

1. 使用默认账户 admin/123456 登录
2. 进入"系统设置"→"用户管理"
3. 修改 admin 用户的密码

#### Q: 上传文件大小限制？

修改 `config/settings.py`：
```python
MAX_CONTENT_LENGTH = 50 * 1024 * 1024  # 默认50MB，可修改
```

#### Q: 如何重置数据库？

删除 `data/app.db` 文件，重启系统会自动重新初始化。
