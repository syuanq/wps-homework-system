# 基于DeepSeek的WPS作业智能批阅与分层推送决策系统

## 使用手册

### 一、系统简介

本系统基于《信息技术基础（WPS Office）》教材开发，能够自动评价学生提交的Word/Excel/PPT作业文件，根据教材操作步骤逐项检查评分，并通过DeepSeek AI大模型根据学生成绩智能生成个性化分层学习任务，实现因材施教。

### 二、技术架构

| 组件 | 技术栈 |
|------|--------|
| 后端框架 | Python Flask |
| 前端框架 | HTML5 + Bootstrap 5 + JavaScript |
| Word解析 | python-docx |
| Excel解析 | openpyxl |
| PPT解析 | python-pptx |
| AI服务 | DeepSeek / 通义千问 / 智谱AI 等 |
| 数据存储 | SQLite |

### 三、默认账户

系统预置了3个演示账户：

| 用户名 | 密码 | 角色 | 说明 |
|--------|------|------|------|
| admin | 123456 | 管理员 | 拥有所有权限，包括系统设置 |
| teacher | 123456 | 教师 | 可管理作业、查看统计、生成教案 |
| student | 123456 | 学生 | 可提交作业、查看评分、获取分层任务 |

### 四、功能说明

#### 4.1 学生端

1. **提交作业**：选择作业任务 → 上传文件 → 系统自动评分
2. **查看评分报告**：查看总分、等级、各操作步骤的通过/未通过详情
3. **获取分层任务**：点击"AI生成分层学习任务"，获取个性化学习建议

#### 4.2 教师端

1. **数据概览**：总提交数、平均分、优秀率、等级分布
2. **作业管理**：查看所有学生提交记录、下载作业、批量导出
3. **学情分析**：各任务平均分、通过率、薄弱技能点排行
4. **教案生成**：根据学情自动生成个性化教案
5. **练习文档**：上传分层练习文档供学生下载

#### 4.3 管理员端（系统设置）

1. **AI配置**：配置API Key、选择AI服务商和模型
2. **用户管理**：管理教师和学生账户
3. **知识库管理**：管理WPS知识点
4. **素材管理**：上传教学素材文件

### 九、常见问题

**Q: 系统启动后页面显示404？**
A: 确保在 `it_homework_system` 目录下运行 `python app.py`。

**Q: 评分结果不准确？**
A: 评分引擎基于文档底层XML结构检测，部分复杂的排版效果可能无法完全识别。可以调整 `scoring_rules.py` 中的检查参数。

**Q: AI分层任务生成失败？**
A: 检查AI配置是否正确。管理员登录后进入"系统设置"→"AI接口配置"，填写API Key。

**Q: 如何添加新的评价任务？**
A: 在 `app/services/scoring_rules.py` 中添加新的任务规则，格式参照现有的 `TASK_RULES`。

**Q: 如何备份数据？**
A: 直接备份 `data/app.db` 文件即可，这是SQLite数据库文件。
